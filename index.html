💡 DCF模型说明

什么是经营价值？

经营价值是指公司在预测期内（如前5年）所有自由现金流的现值总和。它反映了公司核心业务在可预测期间内创造的价值，不包括终值。

DCF计算分解：

公司总价值 = 经营价值 + 终值现值

经营价值 = ∑ [第1年到第n年的现金流现值]

终值现值 = 终值 / (1 + 折现率)^n

核心公式：

公司价值 = ∑ [第t年现金流 / (1 + 折现率)^t] + 终值 / (1 + 折现率)^n

参数说明：

  • 自由现金流: 公司经营活动产生的可自由支配的现金
  • 增长率: 预期现金流的年均增长率
  • 折现率: 反映投资风险的预期回报率
  • 永续增长率: 预测期后的长期稳定增长率
