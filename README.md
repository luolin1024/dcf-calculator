# 公司估值计算器 (DCF模型)

一个基于Vue3的现金流折现模型计算器，用于评估公司的内在价值。

## 功能特点

- 📊 直观的DCF模型计算
- 📈 交互式图表展示
- 📱 响应式设计
- 💰 详细的年度现金流分析
- 🎯 专业的估值参数设置

## 使用方法

1. 在左侧输入区域填写相关参数：
    - 初始自由现金流
    - 预期增长率
    - 折现率
    - 预测年限
    - 永续增长率

2. 点击"开始计算"按钮
3. 查看右侧的计算结果和图表分析

## 技术栈

- Vue 3 (Composition API)
- Chart.js (数据可视化)
- 纯HTML/CSS/JavaScript

## 部署到GitHub Pages

1. 将代码推送到GitHub仓库
2. 在仓库设置中启用GitHub Pages
3. 选择部署分支（通常是main或gh-pages）

## 本地开发

直接在浏览器中打开 `index.html` 文件即可运行。

## 注意事项

本工具计算结果仅供参考，不构成投资建议。实际投资决策应结合更多因素综合分析。
